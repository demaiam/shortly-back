# shortly-back
Um aplicativo whitelabel para encurtar URLS
## Instalação

Clone o repositório:

```bash
git clone https://github.com/demaiam/shortly-back/
```

<br/>

Instale as dependências:

```bash
npm install
```

<br/>

Crie um .env de acordo com o .env.example

<br/>

Crie um banco de dados shortly com o dump

<br/>

Inicie o servidor:

```bash
npm start
```
